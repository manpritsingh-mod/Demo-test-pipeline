# Student Grade Management System

# Demo project for Self-Healing CI/CD Engine

A Python application that manages student records and calculates grades.
Built to demonstrate the Self-Healing CI/CD Engine's ability to detect,
diagnose, and automatically fix build failures.

## Features

- Student CRUD operations
- Grade calculation with weighted categories
- GPA computation
- Honor roll determination
- Grade statistics and analytics

## Running Tests

```bash
pip install -r requirements.txt
python -m pytest tests/ -v
```
